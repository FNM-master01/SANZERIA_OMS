1. システム概要

このシステムは宅配業務に特化した会員管理・伝票印刷システムです。スマホ1台でお客様の登録から伝票印刷まで完結します。

できること

機能説明
会員登録お客様の氏名・住所・電話番号・備考を登録
QRコード発行会員カード用のQRコードを印刷
QRスキャン会員カードのQRを読み取って情報を即座に呼び出し
伝票印刷住所・氏名・メモ・地図QRコード入りの宅配伝票を印刷
複数枚印刷2個口など複数枚の伝票を一度に印刷
クラウド同期Googleスプレッドシートとリアルタイムで同期

使用機器

機器用途
Android スマホ / タブレット（Chrome）メイン操作端末
EPSON TM-m30III-H伝票・QRコード印刷
PC（任意）Googleスプレッドシートで会員管理

2. 起動方法

毎日の起動手順

  1. プリンターの電源を入れる EPSON TM-m30III-Hの電源ボタンを押す
  2. Epson TM Print Assistant を起動する スマホのアプリ一覧から「Epson TM Print Assistant」を起動し、プリンターが接続されていることを確認する
  3. 会員管理システムを開く ホーム画面の「宅配会員管理」アイコンをタップ、または以下のURLをブラウザで開く
起動後の画面

起動すると自動的にGoogleスプレッドシートから最新データを読み込みます。

画面左上の●が●緑なら接続OK、●黄なら未接続です。

💡 TM Print Assistantを先に起動しておかないと印刷できません。毎日必ず先に起動してください。

3. 会員登録・QRコード発行

新規会員を登録する

  1. 「新規登録」タブをタップ 画面上部のタブから「新規登録」を選ぶ
  2. 会員情報を入力する 以下の項目を入力する
項目入力内容必須
姓・名お客様の氏名✅ 必須
電話番号連絡先電話番号✅ 必須
郵便番号配達先郵便番号任意
住所配達先住所（必ず都道府県から入力）任意
区分通常 / 優先 / 法人任意
メモ・備考置き配場所・注意事項など任意
⚠️ 住所は必ず「大阪府吹田市〇〇」のように都道府県から入力してください。都道府県がないと地図QRコードが正確に動作しません。
  1. 「登録してQRコードを発行」をタップ 会員番号が自動採番され、QRコードが表示される
  2. 「QRコードを印刷する」をタップ プリンターからQRコードと会員番号が印刷される
  3. カード台紙にQRを貼り付ける 印刷されたQRコードを名刺サイズのカード台紙に貼り付けてお客様にお渡しする
✅ 登録と同時にGoogleスプレッドシートにも自動保存されます
💡 会員番号は自動で「MB-0001」「MB-0002」…と採番されます。変更はできません。

4. 宅配伝票の印刷

QRコードをスキャンして印刷する

QRスキャン
→
会員情報表示
→
内容確認
→
印刷
  1. 「スキャン」タブをタップ
  2. カメラアイコンをタップしてQRをスキャン 会員カードのQRコードにカメラを向けると自動で読み取られる
  3. 会員情報と印刷プレビューを確認する 氏名・住所・メモが正しいか確認する
  4. 枚数を設定する（通常は1枚） 「−」「＋」ボタンで枚数を調整する
  5. 「TM-m30III-H で印刷」をタップ 自動的にTM Print Assistantが起動して印刷される

💡 伝票下部の地図QRコードをスマホでスキャンするとGoogleマップが開き、そのままナビを開始できます。

会員番号を直接入力して検索する場合

QRカードがない場合は、スキャン画面の入力欄に会員番号（例：MB-0001）を直接入力して「検索」ボタンをタップしてください。

5. 会員一覧の確認・管理

会員一覧を見る

  1. 「会員一覧」タブをタップ 登録されている全会員が一覧表示される
  2. 検索欄で絞り込む（任意） 名前・住所・会員番号・電話番号で検索できる
  3. 会員をタップするとスキャン画面に移動 そのまま伝票印刷ができる

6. 会員情報の編集（スプレッドシート）

住所変更・メモの追記などはPCのGoogleスプレッドシートから直接編集できます。

スプレッドシートで編集する

  1. PCでGoogleスプレッドシートを開く Googleドライブから「会員データ」シートを開く
  2. 該当の行を直接編集する 住所・電話番号・メモなどを修正する
  3. スマホでシステムを再起動する 次回起動時に自動的に最新データが反映される

スプレッドシートの列構成

列項目注意事項
A会員番号変更しないこと（MB-0001形式）
B氏名姓と名の間にスペース（山田 花子）
C電話番号
D郵便番号
E住所都道府県から入力
F区分通常 / 優先 / 法人
Gメモ置き配場所・注意事項など
H登録日自動入力
⚠️ A列の会員番号は絶対に変更しないでください。QRコードと紐付いているため、変更するとQRが読み取れなくなります。

7. QRコードの再発行

会員カードを紛失した場合などにQRコードを再発行できます。

  1. 「スキャン」タブで会員を呼び出す 会員番号を直接入力して検索する
  2. 「🔲 QRコードを再発行」ボタンをタップ 会員情報の下にボタンが表示されている
  3. 「QRコードを印刷する」をタップ 同じ会員番号のQRコードが印刷される
💡 再発行されるQRコードは最初に発行したものと全く同じです。古いカードも引き続き使用できます。

8. 複数枚印刷（複数個口）

2個口・3個口など複数の荷物がある場合、同じ伝票を複数枚まとめて印刷できます。

  1. 通常通り会員をスキャンする
  2. 印刷プレビューの下の「＋」ボタンで枚数を増やす 「−　1　＋」の＋を押して枚数を設定する（最大10枚）
  3. 「TM-m30III-H で印刷」をタップ 設定した枚数分まとめて印刷される
💡 印刷後は自動的に1枚にリセットされます。

設定画面を開く

画面右上の「⚙️ 設定」をタップすると設定画面が開きます。

10. よくあるトラブルと対処法

印刷ボタンを押しても何も起きない

Epson TM Print Assistantが起動しているか確認してください。アプリを起動してプリンターと接続してから再度お試しください。

QRコードを読み取れない

カメラのアクセス許可が必要です。スマホの設定 → アプリ → Chrome → 権限 → カメラ をオンにしてください。また、QRコードが折れ曲がったり汚れている場合は会員番号を直接入力して検索できます。

会員データが表示されない

インターネット接続を確認してください。オフラインの場合は前回取得したデータが表示されます。設定画面の「☁️ スプレッドシートから読み込む」を手動でタップしてください。

地図QRコードを読み取っても正しい場所が出ない

住所が都道府県から入力されていない可能性があります。スプレッドシートで住所を「大阪府〇〇市...」のように都道府県から修正してください。

プリンターが見つからない（TM Print Assistant）

スマホのBluetoothがオンになっているか確認してください。プリンターの電源を入れ直してから再度接続を試みてください。

画面左上の●が黄色になっている

Googleスプレッドシートと未接続の状態です。設定画面でApps Script URLが正しく設定されているか確認してください。

新規登録後「続けて登録する」を押すと前の入力が残っている

画面を上にスクロールすると入力欄がリセットされた状態で表示されます。
