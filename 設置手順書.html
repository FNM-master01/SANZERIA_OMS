宅配会員管理システム — スマホで使えるようにする方法

この手順に沿って進めると、約15〜20分でスマホのホーム画面からアプリのように使えるようになります。費用は完全無料です。

STEP 1 ── GitHubアカウントを作る

1GitHubを開く
スマホまたはPCのブラウザで https://github.com を開きます。
2「Sign up」で無料登録
  • 右上の「Sign up」をタップ
  • メールアドレス・パスワード・ユーザー名を設定
  • 無料プラン（Free）を選択して完了
※ ユーザー名はURLの一部になります。半角英数字で設定してください（例：tanaka-delivery）

STEP 2 ── リポジトリ（保存場所）を作る

1新しいリポジトリを作成
  • ログイン後、右上の「＋」ボタン → 「New repository」をタップ
2設定を入力
  • Repository name（名前）：member と入力
  • Public（公開）を選択（無料で使うために必要）
  • 「Add a README file」にチェックを入れる
  • 「Create repository」ボタンを押す
※ Publicにしても、会員データはスマホ本体に保存されるため、インターネット上には漏れません。

STEP 3 ── HTMLファイルをアップロード

1ファイルをアップロード
  • 作成したリポジトリのページを開く
  • 「Add file」→「Upload files」をタップ
  • ダウンロードした member-system.html を選んでアップロード
2ファイル名を変更（重要）
  • アップロード後、ファイルをクリック → 右上の「✏️（鉛筆アイコン）」をクリック
  • ファイル名を member-system.html から index.html に変更
  • 下にある「Commit changes」ボタンを押して保存
※ 「index.html」にするとURLがシンプルになります。

STEP 4 ── GitHub Pages を有効にする

1Settings を開く
リポジトリのページ上部の「Settings」タブをクリック
2Pages の設定
  • 左メニューの「Pages」をクリック
  • 「Branch」を「main」に変更
  • 「Save」ボタンを押す
3URLが発行される（1〜2分待つ）
数分後にページを更新すると、以下のようなURLが表示されます：
https://あなたのユーザー名.github.io/member/
✓ このURLがあなたの会員管理システムのアドレスです！

STEP 5 ── スマホのホーム画面に追加する

iPhone の場合
  • Safariで発行されたURLを開く
  • 下の「共有」ボタン（四角に上矢印）をタップ
  • 「ホーム画面に追加」をタップ → 「追加」
Android の場合
  • Chromeで発行されたURLを開く
  • 右上の「⋮（メニュー）」をタップ
  • 「ホーム画面に追加」をタップ
✓ アプリアイコンがホーム画面に表示され、タップするだけで起動できます！

よくある質問

会員データはどこに保存される？
スマホのブラウザ内（ローカルストレージ）に保存されます。インターネット上には送信されないため安全です。ただし、同じスマホ・同じブラウザでのみデータが共有されます。
※ ブラウザのキャッシュを全消去するとデータも消えます。定期的にバックアップをお勧めします（将来的にCSVエクスポート機能を追加できます）。
複数のスマホで共有したい場合は？
現在のシステムは各端末独立です。複数端末でデータを共有するには、データベースサーバー（Supabase など無料サービス）との連携が必要です。必要であればご相談ください。
印刷がうまくできない場合は？
TM-m30III-H を同じWiFiネットワークに接続し、プリンターのIPアドレスを設定に追加することで、ワンタップで自動印刷できるようになります。詳細はご相談ください。
