STEP 1 ── GitHub Pages にシステムを設置

1GitHubアカウントを作成
https://github.com を開いて「Sign up」から無料登録
ユーザー名はURLの一部になります（例：tanaka-delivery）
2リポジトリを作成
  • 右上「＋」→「New repository」
  • 名前：member
  • Public（公開）を選択
  • 「Add a README file」にチェック
  • 「Create repository」をクリック
3HTMLファイルをアップロード
  • 「Add file」→「Upload files」
  • member-system.html をアップロード
  • ファイルをクリック → 鉛筆アイコン → ファイル名を index.html に変更
  • 「Commit changes」で保存
4GitHub Pages を有効化
  • 「Settings」→ 左メニュー「Pages」
  • Branch を「main」に変更 → 「Save」
  • 1〜2分後にURLが発行される
https://あなたのユーザー名.github.io/member/
✅ このURLがシステムのアドレスです
5スマホのホーム画面に追加
Android（Chrome）：URLを開く → 右上「⋮」→「ホーム画面に追加」
iPhone（Safari）：URLを開く → 下の共有ボタン →「ホーム画面に追加」

STEP 2 ── Googleスプレッドシート連携

1スプレッドシートを作成
https://docs.google.com/spreadsheets/create を開いて新規作成
2Apps Script を開く
スプレッドシートのメニュー「拡張機能」→「Apps Script」をクリック
3コードを貼り付け
  • 最初から書いてあるコードを全部削除
  • GitHubにある google-apps-script.js を開いてコードを全部コピー
  • Apps Scriptに貼り付けてCtrl+Sで保存
💡 google-apps-script.js はGitHubのリポジトリに保管してあります。いつでもコードを確認・コピーできます。
4デプロイする
  • 「デプロイ」→「新しいデプロイ」
  • 種類：ウェブアプリ
  • アクセスできるユーザー：全員
  • 「デプロイ」→ 権限を承認（Googleアカウントで許可）
  • 発行されたURLをコピーしてどこかにメモしておく
権限承認時に「このアプリは確認されていません」と出たら「詳細」→「安全ではないページへ移動」を選んでください。自分で作ったコードなので安全です。
⚠️ 発行されたApps Script URLは他人に教えないでください。このURLを知っている人は会員データにアクセスできます。
5システムにURLを設定
  • システムを開いて右上「⚙️ 設定」をタップ
  • Apps Script のURLを貼り付け
  • 「URLを保存して接続テスト」をタップ
✅ 接続成功！スプレッドシートに〇件のデータがあります
💡 一度設定すると次回からは自動で接続されます。再設定は不要です。

STEP 3 ── プリンター設定（EPSON TM-m30III-H）

1Epson TM Print Assistant をインストール
Android：Google Play で「Epson TM Print Assistant」を検索
iPhone：App Store で「Epson TM Print Assistant」を検索
2Bluetoothでペアリング
  • プリンターの電源を入れる
  • スマホの「設定」→「Bluetooth」を開く
  • 「TM-m30III-H_XXXXXX」をタップしてペアリング
3TM Print Assistant でプリンターを設定
  • アプリを開く
  • 「プリンター未設定」をタップ
  • 「Bluetooth」→「TM-m30III-H_XXXXXX」を選択
  • 「プリンターの接続を確認する（テスト通信）」で印刷テスト
✅ Print successfully!! と印刷されれば完了
4ePOS-Print を有効化
Epson TM Utility アプリで：
  • 「プリンターの設定を変更する」→「各種設定」
  • 右にスクロール →「インテリジェント機能」をタップ
  • 「ePOS-Device」を無効に変更 →「プリンターへ設定」
  • 「ePOS-Print」を有効に変更 →「プリンターへ設定」

日常の使い方

作業 手順
会員登録 「新規登録」タブ → 情報入力 →「登録してQRコードを発行」→「QRコードを印刷する」でカード用QRを印刷
宅配伝票印刷 TM Print Assistantを起動 →「スキャン」タブ → QRをスキャン →「TM-m30III-H で印刷」
会員情報の編集 PCでGoogleスプレッドシートを開いて直接編集 → スマホ起動時に自動反映
QR再発行 「スキャン」→ 会員を呼び出す →「🔲 QRコードを再発行」→「QRコードを印刷する」

注意事項

⚠️ 住所は必ず都道府県から入力してください。「大阪府吹田市〇〇」のように入力しないと地図QRコードが正確に動作しません。
⚠️ 宅配伝票を印刷する前にEpson TM Print Assistantを起動してプリンターと接続しておいてください。
💡 会員データはGoogleスプレッドシートに自動保存されます。PCでスプレッドシートを開くといつでも確認・編集できます。
💡 システムを開くと自動的に最新データを取得します。手動での同期操作は不要です。
